# String Algorithms Visualizer

An interactive web app for visualizing string algorithms, built with React, TypeScript, Vite, and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+ and npm

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

   The app will be available at the URL printed in the terminal (usually [http://localhost:5173](http://localhost:5173)).

## Available Scripts

- `npm run dev` — start the Vite dev server with hot reload.
- `npm run build` — type-check and create a production build in `dist/`.
- `npm run preview` — preview the production build locally.
